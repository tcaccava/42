# Week 0 - Spaceship Movement Game

## Mandatory Requirements ✅
- ☑️ Spaceship character (please consider it a space taxi)
- ☑️ 4-directional movement with arrow keys
- ☑️ Sprite changes based on movement direction

## Bonus Features ✅
- ☑️ Smooth acceleration/deceleration with inertial system
- ☑️ Custom spaceship sprites
- ☑️ Advanced color-based speed animation (white → yellow → cyan → magenta) with smooth transitions

## How to Play
Use arrow keys to move the spaceship. Watch the color change based on speed! Play on a big screen.
