Webserv is a custom HTTP server built from scratch in C++ as part of the 42cursus project.

🚀 Features:

  • HTTP/1.1 compliant
  • GET, POST, DELETE methods
  • Static file serving
  • CGI support (.py, .php, .pl)
  • File uploads
  • Directory listings
  • HTTP redirections (301, 302, 307, 308)
  • Custom error pages
  • Configurable via .conf files
