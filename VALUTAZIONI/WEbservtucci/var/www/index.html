🚀 Welcome to Webserv

A simple HTTP server made with C++ | 42cursus project

🔧 Features Implemented

  • ✅ GET / POST / DELETE support
  • ✅ Static file serving
  • ✅ Directory listing (autoindex)
  • ✅ CGI support (.py, .php, .pl)
  • ✅ Error handling & custom error pages
  • ✅ File uploads
  • ✅ Redirection (301, 302, 307, 308)
  • ✅ Configurable via .conf file
