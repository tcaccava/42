# Network Training Project

## Description

This project is a networking training interface designed to develop a practical understanding of fundamental networking concepts through progressive levels.

The goal of the project is to configure and troubleshoot network topologies by applying core TCP/IP knowledge. Each level introduces specific challenges related to IP addressing, subnetting, routing, and network communication.

Through this project, students learn how to:
- Configure IPv4 addresses and subnet masks
- Calculate valid host ranges
- Identify network and broadcast addresses
- Configure default gateways
- Understand routing between subnets
- Work with switches and routers
- Apply the OSI and TCP/IP models in practical scenarios

The training is divided into 10 levels, each increasing in complexity and requiring proper configuration export upon completion.
